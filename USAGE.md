# Jarvis Virtual-COS — Руководство пользователя и разработчика

---

## Введение
Jarvis Virtual-COS — интеллектуальный советский ассистент с глубокой когнитивной архитектурой.  
Проект реализует многоуровневую память, адаптивное самообучение и интерактивный веб-интерфейс для быстрого обучения.

---

## Основные компоненты

- **main.py**  
  Запуск веб-панели, TCP-сервера, когнитивного цикла и интеграция модулей.

- **memory/**  
  Хранение и обработка смысловых ассоциаций.

- **interface/**  
  Обработка пользовательских команд и интерактивных вводов.

- **meta/**  
  Meta-слои смыслового поля — `field7d.py`, `field11d.py`, `field15d.py`.

---

## Установка и запуск

1. Клонируйте репозиторий:
git clone https://github.com/KHUSHVAKHT175/Jarvis-Virtual-COS.git
cd Jarvis-Virtual-COS

2. Установка зависимостей (если есть `requirements.txt`):
pip install -r requirements.txt

3. Запуск Jarvis:
python main.py

# Quick Start для пользователя

Добро пожаловать в Jarvis Virtual‑COS! Здесь вы можете экспериментировать с командной панелью и видеть результаты работы системы в реальном времени.

## 1. Вход в веб‑интерфейс
Откройте браузер и перейдите на: http://localhost:5000 (или ваш локальный хост/порт).  
Вы увидите форму для ввода команд и панель с текущими значениями:

- Пульс системы  
- Энергия поля 7D  
- Логи событий

## 2. Основные команды

| Команда             | Что делает                               | Что увидит пользователь                  | Возможности                       |
|---------------------|----------------------------------------|------------------------------------------|----------------------------------|
| `status`            | Показать текущее состояние системы     | Пульс, энергия 7D, количество записей памяти | Следить за «здоровьем» Jarvis     |
| `associate [тема]`  | Создать новую ассоциацию в памяти      | Подтверждение создания ассоциации, увеличение энергии поля | Добавлять новые знания в систему  |
| `analyze [тема]`    | Запросить анализ темы                   | Результаты анализа, связанные ассоциации | Узнавать связи между понятиями    |
| `clear`             | Очистить сессию                        | Пустые поля логов, обнуление текущего ввода | Начать работу с чистого листа      |
| `help`              | Показать список команд                  | Список всех доступных команд             | Быстро ориентироваться в интерфейсе |

*Можно экспериментировать: команды можно комбинировать, и Jarvis будет «накапливать» знания.*

## 3. Что вы увидите

- **Панель пульса:** визуальный индикатор активности системы  
- **Энергия 7D:** число от 0 до 1.00, отражающее степень «полноты знаний» системы  
- **Логи:** все действия и ответы Jarvis в реальном времени  
- **Ассоциации:** новые связи, которые Jarvis создает при ваших командах

## 4. Какие возможности открываются

- Следить за развитием «интуитивного поля» Jarvis  
- Добавлять и изучать ассоциации  
- Получать анализ тем и связей в режиме реального времени  
- Понимать, как система запоминает и обрабатывает ваши команды
